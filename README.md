# Samatva
Post-diagnosis clarity system for women
## From Confusion to Balance

### Problem
Women face confusion after diagnosis due to lack of structured guidance.

### Solution
Samatva provides pattern-based insights and expert-backed guidance through a simple weekly system.

### Features
- Weekly check-in
- Pattern detection
- Insight generation
- Doctor insights hub
- Simple recommendations

### SDG Alignment
- SDG 3: Good Health
- SDG 5: Gender Equality
- SDG 10: Reduced Inequality
